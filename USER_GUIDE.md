## ğŸ†˜ Troubleshooting

### **Common Issues**

#### **Microphone Not Working**
- Check browser permissions for microphone access
- Ensure no other apps are using the microphone
- Try refreshing the page and allowing access again

#### **Story Generation Taking Too Long**
- Our AI typically responds within 10-30 seconds
- If it takes longer, check your internet connection
- Try refreshing and starting over if needed

#### **Can't Edit My Story**
- Make sure you're logged into the correct account
- Check if the story is in "published" status (create a new version to edit)
- Try clearing your browser cache

#### **Export Not Working**
- Ensure pop-ups are allowed in your browser
- Check your download folder for the file
- Try a different export format
